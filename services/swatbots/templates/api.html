Gamboard APIs


Name : Badge
Format : URL /badge/general
Description : get the list of badges; their names, images and description
Sample : api.klicktric.com/badge/general

Profile APIs


Name : Profile
Format : URL /profile/fb_id
Description : fetch klicktric profile info given the fbid of the user.
Sample : api.klicktric.com/profile/638449578
Name : Meter
Format : URL /meter/fb_id?resolution
Description :Fetch the data for a given user
Sample : api.klicktric.com/meter/638449578?daily
Name : Reward
Format : URL /reward/fb_id
Description :Fetch all the points and rewards gained by the user
Sample : api.klicktric.com/reward/638449578

Other APIs


Name : Sync
Format : URL/sync/fb_id?return_code_from_energymolnet
Description : sync the energymolnet account with the users klictric account
Sample : api.klicktric.com/sync/638449578?23423422342rkjewkhwti1iofwlkn
